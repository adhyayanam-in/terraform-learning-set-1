Getting Started

Prerequisites

Everything you need to apply the Terraform code from these modules in your own AWS account.

🔧 Terraform CLI

Version >= 1.5 required. Install via HashiCorp's installer or brew install terraform on macOS.

☁️ AWS Provider

Version ~> 6.0 (currently 6.53.0). Declared in required_providers — Terraform downloads it automatically on terraform init.

🔑 AWS Credentials

Any valid method: environment variables (AWS_ACCESS_KEY_ID), ~/.aws/credentials profile, or an IAM instance profile.

🐍 Python 3.12

Required for the Lambda handler examples. Package each handler into a ZIP file before running terraform apply.

# Minimal provider configuration used across all modules
terraform {
  required_version = ">= 1.5"
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "~> 6.0"
    }
  }
}

provider "aws" {
  region = "us-east-1"

  default_tags {
    tags = {
      Project     = "terraform-aws-learning"
      ManagedBy   = "terraform"
    }
  }
}
